# Веб-приложение для управления вакансиями (Java + Spring Boot)

Это веб-приложение предназначено для сбора, фильтрации и отображения вакансий с сайта HH.ru. Проект реализован на Java (Spring Boot) с использованием PostgreSQL и Docker.

## 1. Описание проекта

Приложение автоматически собирает вакансии с HH.ru, сохраняет их в базу данных и предоставляет удобный веб-интерфейс для поиска, фильтрации и просмотра вакансий.

- Backend: Java 17, Spring Boot, Spring Data JPA, Liquibase
- Frontend: HTML, CSS, JavaScript
- База данных: PostgreSQL
- Контейнеризация: Docker, Docker Compose

## 2. Основные возможности
- Загрузка и обновление вакансий с HH.ru
- Фильтрация по городу, компании, зарплате, типу занятости, ключевым словам в описании
- Поиск по названию или полному тексту вакансии
- Пагинация (постраничный просмотр)
- Просмотр подробной информации о вакансии

## 3. Требования
- Docker и Docker Compose
- (Для локального запуска без Docker) Java 17+, Maven, PostgreSQL

## 4. Быстрый старт (через Docker)

1. Клонируйте репозиторий:
   ```bash
   git clone <repository_url>
   cd hh-parser-main
   ```
2. Запустите приложение:
   ```bash
   docker-compose up --build
   ```
3. После запуска приложение будет доступно по адресу: [http://localhost:8080](http://localhost:8080)

## 5. Запуск в IntelliJ IDEA

1. Откройте папку проекта в IntelliJ IDEA (File > Open...)
2. Дождитесь импорта Maven зависимостей (pom.xml)
3. Убедитесь, что PostgreSQL запущен и параметры подключения указаны в `src/main/resources/application.properties`
4. Запустите класс `VacanciesManagerApplication.java` (правый клик > Run)
5. Откройте [http://localhost:8080](http://localhost:8080) в браузере

## 6. Структура проекта
- `src/main/java` — исходный код приложения
- `src/main/resources` — ресурсы, настройки, миграции Liquibase
- `docker-compose.yml`, `Dockerfile` — контейнеризация
- `README.md` — документация

## 7. Использование приложения
- На главной странице отображается таблица вакансий
- Используйте фильтры и поиск для отбора интересующих вакансий
- Для обновления данных нажмите "Обновить данные"
- Для перехода между страницами используйте пагинацию внизу таблицы

## Примечания
- Источник вакансий: только HH.ru
- Для расширения функционала (добавление новых источников, интеграция с другими API) используйте интерфейс `VacancyParser`
