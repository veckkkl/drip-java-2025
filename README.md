# DRIP Java 2025

Учебный репозиторий с практическими заданиями по Java и итоговым веб-приложением для работы с вакансиями HeadHunter.

## Итоговый проект

### Vacancies Manager

Spring Boot-приложение получает вакансии через API HH.ru, сохраняет их в PostgreSQL и отображает в веб-интерфейсе с фильтрацией и пагинацией.

**Стек:** Java 17, Spring Boot 3, Spring Data JPA, WebClient, Thymeleaf, Liquibase, PostgreSQL, Docker Compose.

Основные возможности:

- загрузка вакансий с HH.ru;
- фильтрация по названию, городу, компании, зарплате и типу занятости;
- постраничный вывод результатов;
- хранение данных в PostgreSQL;
- автоматическое создание схемы базы данных через Liquibase;
- запуск приложения и базы данных одной командой.

### Быстрый запуск

Для запуска потребуются Docker и Docker Compose.

```bash
git clone git@github.com:veckkkl/drip-java-2025.git
cd drip-java-2025/PROJECT/hh-parser-main
docker compose up --build
```

После запуска интерфейс доступен по адресу [http://localhost:8080](http://localhost:8080).

Остановить контейнеры:

```bash
docker compose down
```

Удалить контейнеры вместе с данными PostgreSQL:

```bash
docker compose down -v
```

Подробное описание архитектуры, конфигурации и локального запуска находится в [документации Vacancies Manager](PROJECT/hh-parser-main/README.md).

## Учебные задания

| Раздел | Темы |
| --- | --- |
| `homework1` | базовые алгоритмы, строки, массивы, числа |
| `task1` | представление и вычисление математических выражений |
| `task2` | динамический массив целых чисел |
| `task3` | интерфейсы, исключения и управление соединениями |
| `task4` | анализ стека вызовов |
| `homework3` | коллекции, итераторы, сортировка, преобразование данных |
| `homework4/homework4` | даты, регулярные выражения, файлы, потоки и NIO |
| `homework5` | многопоточность, синхронизация, блокировки и Fork/Join |
| `PROJECT/hh-parser-main` | итоговое Spring Boot-приложение |

Учебные задания представлены отдельными Java-файлами и не объединены общим Maven- или Gradle-проектом. Их удобнее запускать по отдельности из IDE с учётом объявленного в файле пакета.

## Структура репозитория

```text
.
├── homework1/                 # базовые задачи
├── homework3/                 # коллекции и обработка данных
├── homework4/homework4/       # даты, файлы и потоки
├── homework5/                 # многопоточность
├── task1/ ... task4/          # отдельные учебные задания
└── PROJECT/hh-parser-main/    # итоговое приложение
```

## Требования

- JDK 17;
- Maven 3.8+ — для итогового проекта;
- Docker и Docker Compose — для рекомендованного запуска;
- PostgreSQL — только при локальном запуске без Docker.
